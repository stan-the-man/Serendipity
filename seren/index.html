About Serendipity




Serendipity is a web based musical search and recommendation system which classifies and recommends music by processing extracted traits from songs. Unlike other such systems which use human based music classification, we have automated the recommendation process and created algorithms to do the work so our users discover new music.
This senior design project of our was intended to be a fun way of exploring and discovering new music.
